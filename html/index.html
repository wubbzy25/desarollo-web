Lenguaje de Marcas de Hipertexto

html sirve principalmente para dar estructura a nuestra pagina colocar los titulos, los texto, las imagenes, y todo el contenido que necesitemos mostrar.

¿De qué va HTML5?

HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros

Sintaxis HTML

HTML es fácil de aprender. Su sintaxis está basada en etiquetas: una de apertura y una de cierre. Y dentro de ellas va el contenido.

ETIQUETAS

HTML cuenta con una gran variedad de etiqueta en las cuales nos ayudan a darle forma a nuestra pagina entre ellas se encuentran title,video,img y etc

HTML NO TIENE BUGS

Al HTML ser un lenguaje de marcado Hipertexto no hay algo que diga que da error, si algo no te sale como esperabas o no sale el navegador nunca te lo marcara como error

SEO

Un buena estructura y sintaxis del codigo limpia y ordena puede ayudar a posicionar tu pagina cada vez mas arriba para que cuando los usuario la busquen siempre este en las posiciones mas favorable esto se conoce como SEO
